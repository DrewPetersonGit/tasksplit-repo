# TaskSplit ✨

Collaborative task management web application built with React, Supabase, and Azure.

## Features
- User authentication
- Project creation
- Task assignment
- Team collaboration

## Tech Stack
- React + Vite
- Supabase
- Azure Static Web Apps

## Live Website
https://www.task-split.app

## Running Locally

```bash
npm install
npm run dev
```
